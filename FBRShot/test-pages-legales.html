🍪 Système de Gestion des Cookies

Bannière de Consentement À tester

La bannière doit apparaître lors de la première visite. Elle permet d'accepter, refuser ou personnaliser les cookies.

Comment tester :
  1. Ouvrez index.html en navigation privée
  2. La bannière doit apparaître en bas de page
  3. Testez les 3 boutons : Accepter / Refuser / Personnaliser
  4. Vérifiez que votre choix est mémorisé (cookies)

⚙️ Configuration Requise

Google Analytics 4 Action requise

Étapes :

  1. Créez un compte Google Analytics sur analytics.google.com
  2. Créez une propriété GA4
  3. Récupérez votre Measurement ID (format : G-XXXXXXXXXX)
  4. Ouvrez le fichier assets/js/cookies.js
  5. Ligne 13, remplacez GA_ID: 'G-XXXXXXXXXX' par votre ID
  6. Sauvegardez et testez
