# Dossier Photos Photographe

## 📸 Instructions pour ajouter vos photos

Ce dossier contient les photos qui seront affichées dans la section "Service Photographe" de votre site.

### Photos requises

Ajoutez **5 photos** dans ce dossier avec les noms suivants :

1. `photo1.jpg` - Photo d'événement mariage
2. `photo2.jpg` - Photo d'événement corporate
3. `photo3.jpg` - Photo d'événement anniversaire
4. `photo4.jpg` - Photo d'événement soirée
5. `photo5.jpg` - Photo d'événement festival

### Format recommandé

- **Format** : JPG ou PNG
- **Orientation** : Portrait (vertical) recommandé pour un meilleur rendu
- **Dimensions recommandées** :
  - Largeur : 600-1000px
  - Hauteur : 800-1333px
  - Ratio : 3:4 (portrait)
- **Poids** : Maximum 500KB par image (compressez si nécessaire)

### Optimisation des images

Pour de meilleures performances, utilisez des outils comme :
- **TinyPNG** : https://tinypng.com/
- **Squoosh** : https://squoosh.app/
- **ImageOptim** (Mac) : https://imageoptim.com/

### Exemple de nommage

```
photographer/
├── photo1.jpg  (Couple de mariés)
├── photo2.jpg  (Événement d'entreprise)
├── photo3.jpg  (Fête d'anniversaire)
├── photo4.jpg  (Soirée festive)
└── photo5.jpg  (Festival/Concert)
```

### Fallback

Si une photo n'est pas présente, un placeholder avec une icône 📸 s'affichera automatiquement.

### Alternative : Format WebP

Pour de meilleures performances, vous pouvez aussi fournir les images en format WebP :

```
photographer/
├── photo1.jpg
├── photo1.webp
├── photo2.jpg
├── photo2.webp
...
```

Le navigateur chargera automatiquement le format le plus approprié.

### Conseils de sélection

Pour de meilleurs résultats, choisissez des photos :
- ✅ Bien éclairées et nettes
- ✅ Représentatives de différents types d'événements
- ✅ Montrant des moments authentiques et émouvants
- ✅ De haute qualité professionnelle
- ✅ Avec des personnes (visages floutés si besoin pour RGPD)

### Vérification

Après avoir ajouté vos photos :
1. Ouvrez index.html dans votre navigateur
2. Naviguez vers la section "Service Photographe"
3. Vérifiez que toutes les photos s'affichent correctement
4. Testez sur mobile et tablette

---

**Note** : Si vous n'avez pas encore vos 5 photos, vous pouvez commencer avec 3-4 photos. Les emplacements vides afficheront un placeholder élégant.
