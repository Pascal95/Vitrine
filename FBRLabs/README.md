## 🔧 Configuration

### 1. Activer FormSubmit

Le formulaire de contact est déjà configuré. Pour l'activer :

1. Ouvrez `index.html`
2. Remplissez et envoyez le formulaire une première fois
3. Confirmez l'email que vous recevrez de FormSubmit
4. Tous les prochains formulaires seront envoyés automatiquement à `contact@fbrlabs.fr`

📄 **Guide complet** : Voir `FORMSUBMIT-ACTIVATION.md` (à créer si nécessaire)

### 2. Configurer Google Analytics

1. Créez un compte Google Analytics
2. Obtenez votre Measurement ID (G-XXXXXXXXXX)
3. Ouvrez `assets/js/cookies.js`
4. Ligne 13, remplacez `G-XXXXXXXXXX` par votre ID

### 3. Ajouter vos images

Placez vos images dans `assets/images/` et mettez à jour les références dans `index.html`.
