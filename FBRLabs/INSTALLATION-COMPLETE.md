## 🚀 Prochaines étapes

### 1. Ajouter vos images (RECOMMANDÉ)

Placez vos images dans `assets/images/` :
- Logo FBR Labs
- Images de projets pour le portfolio
- Images d'illustration pour les services

Puis mettez à jour les références dans `index.html`.

### 2. Activer FormSubmit (OBLIGATOIRE)

📧 **Pour recevoir les emails du formulaire de contact** :

1. Ouvrez `index.html` dans un navigateur
2. Remplissez et envoyez le formulaire
3. Confirmez l'email de FormSubmit dans votre boîte `contact@fbrlabs.fr`

📄 **Guide détaillé** : `FORMSUBMIT-ACTIVATION.md`

### 3. Configurer Google Analytics (OPTIONNEL)

Pour le tracking et le SEO :

1. Créez un compte Google Analytics
2. Obtenez votre ID (G-XXXXXXXXXX)
3. Modifiez `assets/js/cookies.js` ligne 13
4. Remplacez `G-XXXXXXXXXX` par votre ID

### 4. Personnaliser le contenu (RECOMMANDÉ)

Adaptez le contenu à votre image :
- Modifiez les textes dans `index.html`
- Ajoutez vos vrais projets au portfolio
- Mettez à jour les tarifs si nécessaire
- Ajoutez vos témoignages clients

### 5. Déployer le site (MISE EN PRODUCTION)

#### Option 1 : OVHcloud (recommandé)
1. Connectez-vous à votre espace OVHcloud
2. Allez dans "Hébergement web"
3. Uploadez tous les fichiers via FTP
4. Configurez le nom de domaine `www.fbrlabs.fr`
5. Activez le HTTPS (certificat SSL)

#### Option 2 : Autres hébergeurs
- Netlify (gratuit, simple)
- Vercel (gratuit, moderne)
- GitHub Pages (gratuit, pour développeurs)

### 6. Soumettre à Google (APRÈS DÉPLOIEMENT)

1. Créez un compte Google Search Console
2. Ajoutez votre site `www.fbrlabs.fr`
3. Soumettez le sitemap : `https://www.fbrlabs.fr/sitemap.xml`
4. Vérifiez l'indexation
